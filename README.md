## 📌 주요 기능

- 🔐 **Firebase Auth 연동**: 로그인 / 로그아웃 구현  
- 🛒 **Realtime Database**: 상품 목록, 장바구니, 관리자 데이터 저장  
- 📷 **Cloudinary API**: 이미지 업로드 및 저장  
- ⚡ **React Query**: 상품 및 장바구니 관련 이벤트 처리  
- 🧭 **React Router**: 페이지 간 라우팅 처리
